How to Play

Use the Left Arrow key to move the paddle left, and the Right Arrow key to move it right.

When the game starts, a ball bounces around. Your job is to move the paddle so the ball never falls below the bottom — you need to catch it every time.

The ball hits bricks at the top; each time the ball hits a brick, that brick disappears and you earn points.

Objective & Winning

Goal: Break every brick on screen by bouncing the ball off your paddle so it hits every brick.

Winning: When all bricks are destroyed → game ends (you win), and your final score displays.

Losing: If the ball misses the paddle and falls past it → game ends (you lose).
